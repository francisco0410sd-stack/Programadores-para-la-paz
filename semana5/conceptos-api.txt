Esta ruta POST llamada /reporte permite que un usuario envíe información al servidor. El servidor recibe el mensaje desde req.body.mensaje y responde con un objeto JSON indicando que el reporte fue recibido junto con el contenido enviado. En una aplicación comunitaria, esta ruta podría servir para enviar reportes de problemas, alertas, solicitudes o información importante. Por ejemplo, los ciudadanos podrían reportar daños en vías, emergencias o necesidades de la comunidad, permitiendo que la información se reciba y organice de manera más rápida y eficiente.
