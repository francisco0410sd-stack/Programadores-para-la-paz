Saber usar la terminal es importante para un desarrollador porque permite trabajar de forma más rápida y eficiente dentro de un proyecto. Con la terminal puedo crear carpetas, organizar archivos y navegar entre directorios sin depender de interfaces gráficas. Además, muchas herramientas de programación funcionan directamente desde la terminal, como Git. Esto facilita el control de versiones y el trabajo en equipo. También ayuda a automatizar tareas repetitivas. Aprender a usar la terminal mejora mi lógica y comprensión del sistema. En esta actividad entendí mejor cómo moverme dentro de un proyecto y crear archivos desde cero. Esto es una base importante para seguir aprendiendo programación.
